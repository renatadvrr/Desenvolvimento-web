O café é uma bebida produzida a partir dos grãos torrados do fruto do cefeeiro. É servido tradicionalmente quente, mas também pode ser consumido gelado. Ele é um estimulante, por possuir cafeína - geralmente 80 a 140 mg para cada 207 ml deoendendo do método de preparação.
